Tutorial Básico de Bootstrap

Bootstrap 4 é a versão mais recente do Bootstrap , que é a estrutura HTML, CSS e JavaScript mais popular para o desenvolvimento de sites móveis responsivos.

Este tutorial do Bootstrap 4 contém algumas das suas funcionalidades mais básicas.

1 - Bootstrap Grid

O sistema de grade do Bootstrap é construído com flexbox e permite até 12 colunas na página. Se não quiser usar todas as 12 colunas individualmente, você pode agrupar as colunas para criar colunas mais largas: Certifique-se de que a soma some 12 ou menos (não é necessário que você use todas as 12 colunas disponíveis).

1.1 - Classes de grid:

  • O sistema de grade Bootstrap 4 tem cinco classes de grid:
  • .col- :dispositivos extra pequenos - largura da tela menor que 576 px
  • .col-sm- :dispositivos pequenos - largura da tela igual ou superior a 576 px
  • .col-md- :dispositivos médios - largura da tela igual ou superior a 768 px
  • .col-lg- :dispositivos grandes - largura da tela igual ou superior a 992 px
  • .col-xl- :dispositivos xlarge - largura da tela igual ou superior a 1200 px

1.2 - Estrutura Básica de um Grid Bootstrap 4:

1.2.1:

Crie uma linha 'div class="row"'. Em seguida, adicione o número desejado de colunas (tags com .col-*-*classes apropriadas ). A primeira estrela (*) representa a responsividade: sm, md, lg ou xl, enquanto a segunda estrela representa um número, que deve somar 12 para cada linha.

1.2.2

Em vez de adicionar um número a cada col, deixe o bootstrap lidar com o layout, para criar colunas de largura igual: dois "col"elementos = 50% de largura para cada coluna. três colunas = 33,33% de largura para cada coluna. quatro .col-sm|md|lg|xlcolunas = largura de 25%, etc. Você também pode usar para tornar as colunas responsivas.

2.2 - Espaçamento, Margem, Padding:

As classes são definidas usando o formato {propriedade}{lados}-{tamanho} para o breakpoint xs e {proprieade}{lados}-{breakpoint}-{tamanho} para sm, md, lg e xl.

  • Bootstrap tem um grande leque de atalhos responsivos de margem e padding para modificar a aparência de um elemento(Deixe o campo em branco, para usar margin ou padding em todos os quatro lados do elemento:
  • .m- : para usar margin;
  • .p- : para usar padding;
  • .mt/.pt- : para usar margin-top ou padding-top;
  • .mb/pb- : para usar margin-bottom ou padding-bottom;
  • .ml/pl- : para usar margin-left ou padding-left;
  • .mr/pr- : para usar margin-right ou padding-right;
  • .mx/px- : para usar margem ou padding, tanto na esquerda quanto direita;
  • .my/py- : y - para usar margem ou padding, tanto na parte superior quanto na inferior;
  • O campo tamanho pode ser:
  • .-0 : para remover o padding ou margin;
  • .-1 : define marginou paddingpara .25rem(4px se o tamanho da fonte for 16px);
  • .-2 : define marginou paddingpara .5rem(8px se o tamanho da fonte for 16px);
  • .-3 : define marginou paddingpara 1rem(16px se o tamanho da fonte for 16px);
  • .-4 : define marginou paddingpara 1.5rem(24px se o tamanho da fonte for 16px);
  • .-5 : define marginou paddingpara 3rem(48px se o tamanho da fonte for 16px);
  • .-auto : pdefine margin para automático

Nota: as margens também podem ser negativas, adicionando um "n" na frente do tamanho.

O Bootstrap 4 tem algumas classes contextuais que podem ser usadas para fornecer "significado por meio de cores":

  • O campo tamanho pode ser:
  • .text-muted : Este texto está desativado;
  • .text-primary : Este texto é importante;
  • .text-success: Este texto indica sucesso;
  • .text-info : Este texto representa alguma informação;
  • .text-warning : Este texto representa um aviso;
  • .text-danger : Este Texto representa perigo;
  • .text-secondary : Texto Secundário
  • .text-dark : Este texto é cinza escuro;
  • .text-body : Cor padrão do corpo (geralmente preto);
  • .text-light: Este texto é cinza claro em background escuro;

4.2 - Dropdown:

Um Dropdown é um menu alternável que permite ao usuário escolher um valor de uma lista predefinida:

A classe '.dropdown' indica um dropdown. Para abrir o dropdown, use um botão ou link com uma classe de .dropdown-toggle e o data-toggle="dropdown" como atributo. Adicione a classe .dropdown-menu a um 'div' elemento para realmente construir o dropdown. Em seguida, adicione a .dropdown-itemclasse a cada elemento (links ou botões) dentro do dropdown.
